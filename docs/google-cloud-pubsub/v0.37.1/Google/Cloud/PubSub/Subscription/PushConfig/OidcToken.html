#audience ⇒ String

Audience to be used when generating OIDC token. The audience claim identifies the recipients that the JWT is intended for. The audience value is a single case-sensitive string.

Having multiple values (array) for the audience field is not supported.

More info about the OIDC JWT token audience here: https://tools.ietf.org/html/rfc7519#section-4.1.3

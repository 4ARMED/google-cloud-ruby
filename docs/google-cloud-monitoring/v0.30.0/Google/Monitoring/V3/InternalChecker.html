Overview

An internal checker allows uptime checks to run on private/internal GCP resources.

#name ⇒ String

Returns A unique resource name for this InternalChecker. The format is:

projects/[PROJECT_ID]/internalCheckers/[INTERNAL_CHECKER_ID].

PROJECT_ID is the stackdriver workspace project for the uptime check config associated with the internal checker.

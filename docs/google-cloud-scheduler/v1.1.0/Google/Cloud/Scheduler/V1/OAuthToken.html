Overview

Contains information needed for generating an OAuth token. This type of authorization should be used when sending requests to a GCP endpoint.

#scope ⇒ String

Returns OAuth scope to be used for generating OAuth access token. If not specified, "https://www.googleapis.com/auth/cloud-platform" will be used.

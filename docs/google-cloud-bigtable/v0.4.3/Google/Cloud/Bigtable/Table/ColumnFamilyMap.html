Overview

Table::ColumnFamilyMap is a hash accepting string ColumnFamily names as keys and GcRule objects as values. It is used to create an instance.

Examples:

Add column family with name and garbage collection rule


column_families = Google::Cloud::Bigtable::Instance::ColumnFamilyMap.new

column_families.add('cf1', Google::Cloud::Bigtable::GcRule.max_versions(3))
